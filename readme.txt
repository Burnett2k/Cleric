To get this project working:

Install GitBash
Install Node.js (so you can run npm commands and start your server)
Install MongoDB


Pull down all the code using:
git clone https://github.com/Burnett2k/Cleric.git

run npm install on the package.json file to install dependencies

Run a MongoDB instance my running mongod.exe in cmd

Run your node server by navigating to the root folder of the app an typing 'node server.js' or if you have nodemon setup use 'nodemon server.js'
